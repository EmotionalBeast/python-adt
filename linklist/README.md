## 无序链表 linklist
    数据区域：存放节点数据
    指针区域：存放指向下一个节点的指针
    无序链表的操作：
        List()
        add(any)
        remove(any)
        exist(item) -> bool
        is_empty() -> bool
        size() -> int
        append(item)  # 末端添加
        index(item) -> int
        insert(pos, item)
        pop()
        pop(pos)
        


## 有序链表 orderedlist
    链表中的数据都有位置
    有序链表的操作
        OrderedList()
        add(item)
        remove(item) -> item
        exist(item) -> bool
        is_empty() -> bool
        size() -> int
        index(item) -> int
        pop() -> item
        pop(pos) -> item
